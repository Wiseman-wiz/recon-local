# BANK RECONCILIATION SYSTEM PHASE 1

A financial management tool designed to streamline the reconciliation process for multiple company bank accounts. It allows users to efficiently manage bank accounts, ensuring accurate financial tracking. The system enables the uploading of General Ledger (GL) and Bank Statements (BS), automating data matching and reducing manual errors. Users can also manage financial records, track transactions, and ensure consistency between company books and bank records. A key feature of the system is the Reconciliation Report, which compares GL and BS data, identifying discrepancies and unmatched transactions to facilitate accurate financial reporting.

Go to [Wiki](https://github.com/BDC-DEV-PH/bank-reconciliation-system/wiki) for more info.
